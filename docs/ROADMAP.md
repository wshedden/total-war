# Roadmap

## v0 completion criteria
- Fast canvas world map with pan/zoom + robust hover picking
- Deterministic turn loop and stable save/load
- Political and heatmap overlays
- Country dossier, search, help, and debug tools
- Automated data refresh/build pipeline with caching

## v1 ideas
- Army units, movement, and supply network
- Deeper diplomacy (treaties, blocs, sanctions)
- Trade lanes and resource production chains
- Scenario setup tools and mod-friendly data packs
- Optional multiplayer lockstep with deterministic replay
